# c2r
Miscellaneous scripts to help processing cryoSPARC results in RELION.

# Requirements
* Python 3

# Installation
```bash
git clone https://github.com/kttn8769/c2r.git
```

# How to use
See help message of each script by invoking with --help option.
```bash
python3 <path to c2r>/scripts/<script name> --help
```

For example
```bash
python3 ~/softwares/c2r/scripts/c2r_prep_star_for_polish.py --help
```
